# FitnesCenter REST API

## Технологии
- **Язык:** C# (.NET 8)
- **Фреймворк:** ASP.NET Core Web API
- **Хранилище:** In-Memory (ConcurrentDictionary)
- **Документация:** Swagger

## Структура проекта
FitnesCenter/
├── FitnesCenter.API/ # Контроллеры, DTO, Middleware
├── FitnesCenter.Application/ # Сервисы (бизнес-логика)
├── FitnesCenter.Domain/ # Сущности и интерфейсы
├── FitnesCenter.Infrastructure/ # In-Memory репозитории
└── FitnesCenter.Shared/ # Enums, Exceptions

## Запуск
```bash
dotnet restore
dotnet run --project FitnesCenter.API

## API Эндпоинты

### Клиенты (`/api/clients`)

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/clients` | Создать клиента |
| GET | `/api/clients` | Все клиенты |
| GET | `/api/clients/{id}` | Клиент по ID |
| GET | `/api/clients/{id}/detail` | Детально (с тренером) |
| PUT | `/api/clients/{id}` | Обновить клиента |
| PATCH | `/api/clients/{id}/status` | Сменить статус |
| POST | `/api/clients/{clientId}/trainer/{trainerId}` | Назначить тренера |

### Тренеры (`/api/trainers`)

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/trainers` | Создать тренера |
| GET | `/api/trainers` | Все тренеры |
| GET | `/api/trainers/{id}/detail` | Детально (с клиентами) |
| PUT | `/api/trainers/{id}` | Обновить тренера |
| PATCH | `/api/trainers/{id}/status` | Сменить статус |

## 🔧 Коды ответов

| Код | Описание |
|-----|----------|
| 200 | Успешный запрос |
| 201 | Ресурс создан |
| 204 | Успешный запрос (без содержимого) |
| 400 | Ошибка валидации |
| 404 | Ресурс не найден |
| 409 | Конфликт (бизнес-правило нарушено) |
| 500 | Внутренняя ошибка сервера |
